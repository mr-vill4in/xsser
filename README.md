# xsser

#1st Step :

git clone https://github.com/mr-vill4in/xsser.git

cd xsser

pip3 install -r req.txt

python3 xsser.py -h

  -u URL          target url
  -urls URL_FILE  target urls


python3 xsser.py -u your_url.php?id= 

or 

python3 xsser.py -urls your_file.txt
